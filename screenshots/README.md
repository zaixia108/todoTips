# Screenshots

本目录用于存放应用程序截图。

## 建议截图内容

1. **主界面**
   - main_window.png - 显示三个标签页（本日、本周、本月）
   
2. **系统托盘**
   - tray_icon.png - 系统托盘图标和菜单
   
3. **添加待办**
   - add_dialog.png - 添加待办对话框
   
4. **设置界面**
   - settings.png - 快捷键和通知设置
   
5. **通知示例**
   - notification.png - 系统通知提醒

## 如何添加截图

1. 运行应用程序
2. 截取相应界面
3. 保存为PNG格式
4. 放置在此目录
5. 在README.md中引用

示例：
```markdown
![主界面](screenshots/main_window.png)
```
