# TodoTips 快速开始指南

## 第一次使用

1. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

2. **启动应用**
   
   Linux/macOS:
   ```bash
   python3 main.py
   # 或使用启动脚本
   chmod +x start.sh
   ./start.sh
   ```
   
   Windows:
   ```bash
   python main.py
   # 或双击 start.bat
   ```

3. **首次设置**
   - 应用会在系统托盘显示图标
   - 默认快捷键已配置好，可以直接使用
   - 右键托盘图标可以访问菜单

## 快速教程

### 场景1: 快速记录想法

1. 当你有一个想法时，将其复制到剪贴板
   例如：`给客户发送项目进度报告`

2. 按下 `Ctrl+Shift+A`

3. 完成！待办已添加，系统会在5分钟、15分钟、1小时后提醒你

### 场景2: 规划一周任务

1. 按下 `Ctrl+Shift+T` 打开主窗口

2. 切换到"本周"标签

3. 点击"+ 添加待办"按钮

4. 输入任务信息：
   - 标题：`准备周五的团队会议PPT`
   - 描述：`包括Q1总结和Q2规划`
   - 类别：本周

5. 点击确定

### 场景3: 导出待办给AI助手

1. 打开主窗口 (`Ctrl+Shift+T`)

2. 点击"导出"按钮

3. 保存为 `todos.json`

4. 将文件内容复制给AI助手，例如：
   ```
   请帮我分析这些待办事项，给出优先级建议：
   [粘贴JSON内容]
   ```

## 常用操作

### 完成待办
- 勾选待办项前的复选框即可标记为完成
- 已完成的待办会显示删除线

### 删除待办
- 右键点击待办项
- 选择"删除"
- 确认删除

### 修改快捷键
1. 右键托盘图标 → 设置
2. 修改快捷键（格式：ctrl+shift+字母）
3. 保存并重启应用

## 提示技巧

1. **批量添加待办**
   - 将多个待办事项放在剪贴板
   - 每次按快捷键会添加当前剪贴板内容
   - 适合快速转录会议记录

2. **利用提醒功能**
   - 使用快捷键添加的待办会自动设置提醒
   - 手动添加的待办需要在描述中注明重要性

3. **定期导出**
   - 每周末导出待办，用于周总结
   - 将数据提交给AI做时间管理分析

4. **分类使用建议**
   - 本日：紧急且必须今天完成的
   - 本周：本周计划，可灵活安排时间
   - 本月：长期目标，月度跟踪

## 疑难解答

**Q: 快捷键不工作？**
A: 
- 检查是否有其他程序占用了该快捷键
- Linux用户可能需要sudo权限
- 尝试修改为其他快捷键组合

**Q: 通知不显示？**
A:
- Windows: 检查通知中心设置
- macOS: 系统偏好设置 → 通知
- Linux: 确认安装了libnotify

**Q: 数据保存在哪里？**
A: 待办数据保存在应用目录下的 `todos.json` 文件中

**Q: 可以在多台电脑同步吗？**
A: 可以将 `todos.json` 文件放在云盘目录中，实现简单同步

## 进阶用法

### 配合AI使用

导出的JSON格式非常适合AI分析，可以问AI：

- "帮我按优先级重新排序这些任务"
- "估算完成这些任务需要多少时间"
- "哪些任务可以合并或委派"
- "根据任务类型给出时间规划建议"

### 自定义样式

可以修改 `main_window.py` 中的 `apply_stylesheet()` 方法来自定义界面颜色和样式。

### 扩展功能

项目采用模块化设计，可以轻松扩展：

- 添加新的提醒时间模式
- 支持标签系统
- 添加任务优先级
- 支持子任务
- 添加番茄钟功能

欢迎贡献代码！
