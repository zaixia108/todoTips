# TodoTips - 快速待办事项管理工具

一个使用 PySide6 开发的现代化待办事项管理工具，支持快捷键快速添加、系统通知提醒、后台运行等功能。

## 功能特性

### 基本功能
1. **快捷键支持**
   - 快捷键1 (默认: Ctrl+Shift+A): 快速从剪贴板添加待办事项
   - 快捷键2 (默认: Ctrl+Shift+T): 打开主界面
   - 可在设置中自定义快捷键

2. **智能提醒**
   - 使用快捷键添加的待办会自动设置三次提醒
   - 提醒时间: 5分钟后、15分钟后、1小时后
   - 系统通知 + 音效提醒

3. **多板块管理**
   - 本日待办：当天需要完成的任务
   - 本周待办：本周计划
   - 本月待办：长期规划

### 附加功能
1. **后台运行**
   - 最小化到系统托盘
   - 托盘菜单快速访问

2. **系统通知**
   - 跨平台系统通知支持
   - 可选音效提醒
   - 待办到期自动提醒

3. **数据导出**
   - 导出所有待办为JSON格式
   - 方便提交给LLM进行任务分析和汇总
   - 支持按板块查看统计信息

4. **现代化UI**
   - 简洁美观的界面设计
   - 响应式布局
   - 清晰的视觉层次

## 安装说明

### 系统要求
- Python 3.6+
- Windows / macOS / Linux

### 安装依赖

```bash
pip install -r requirements.txt
```

依赖包包括：
- PySide6: Qt图形界面框架
- keyboard: 全局快捷键支持
- pyperclip: 剪贴板操作
- playsound: 音效播放

### Linux 额外依赖

在 Linux 系统上，可能需要安装额外的系统包：

```bash
# Ubuntu/Debian
sudo apt-get install python3-xlib
sudo apt-get install libnotify-bin  # 系统通知

# Fedora/RHEL
sudo dnf install python3-xlib
sudo dnf install libnotify  # 系统通知
```

## 使用方法

### 启动应用

```bash
python main.py
```

**注意**: 在某些Linux系统上，可能需要使用sudo运行以支持全局快捷键：

```bash
sudo python main.py
```

### 基本操作

#### 1. 快速添加待办（推荐方式）
1. 复制你想添加的待办事项文本
2. 按下快捷键 `Ctrl+Shift+A`
3. 待办事项会自动添加到"本日"列表
4. 系统会在 5分钟、15分钟、1小时后提醒你

#### 2. 手动添加待办
1. 按下快捷键 `Ctrl+Shift+T` 打开主窗口
2. 或者从系统托盘双击图标
3. 点击"+ 添加待办"按钮
4. 填写标题、描述（可选）
5. 选择类别（本日/本周/本月）
6. 点击确定

#### 3. 管理待办
- **完成待办**: 勾选待办项前的复选框
- **删除待办**: 右键点击待办项，选择"删除"
- **切换板块**: 点击顶部的"本日"、"本周"、"本月"标签

#### 4. 导出待办
1. 点击"导出"按钮
2. 选择保存位置和文件名
3. 导出的JSON文件包含所有待办信息和统计数据

#### 5. 自定义设置
1. 从系统托盘菜单选择"设置"
2. 自定义快捷键组合
3. 启用/禁用音效
4. 保存设置（快捷键需要重启生效）

### 导出数据格式

导出的JSON文件结构如下：

```json
{
  "today": {
    "category": "today",
    "export_time": "2026-02-13T01:00:00",
    "total": 5,
    "completed": 2,
    "todos": [...]
  },
  "week": {...},
  "month": {...}
}
```

可以直接将此文件提交给LLM进行任务分析、优先级排序、时间规划等。

## 快捷键说明

默认快捷键：
- `Ctrl+Shift+A`: 快速添加剪贴板内容为待办
- `Ctrl+Shift+T`: 显示主窗口

快捷键格式规则：
- 支持的修饰键: ctrl, shift, alt
- 可以组合使用，如: ctrl+shift+字母
- 避免与系统快捷键冲突

## 数据存储

- 待办数据保存在 `todos.json` 文件中
- 设置保存在系统配置目录
- 应用关闭时自动保存所有数据

## 故障排除

### 快捷键不工作
1. 确认快捷键没有被其他程序占用
2. Linux系统可能需要使用sudo运行
3. 检查设置中的快捷键配置是否正确

### 通知不显示
- Windows: 检查系统通知设置
- macOS: 授予应用通知权限
- Linux: 确认已安装 libnotify

### 音效不播放
- Windows: 检查系统音量设置
- macOS/Linux: 确认音频服务正常运行
- 在设置中确认已启用音效

## 技术栈

- **UI框架**: PySide6 (Qt for Python)
- **快捷键**: keyboard库
- **剪贴板**: pyperclip
- **通知**: 系统原生通知API
- **数据存储**: JSON
